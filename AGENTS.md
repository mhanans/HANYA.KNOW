# Repository-wide Instructions

- Semua dokumentasi terkait modul Pre-Sales harus mencantumkan urutan proses terbaru: `Presales Workspace → Timeline Estimator → Timeline Generation → Estimated Cost Generation`.
- Jelaskan bahwa Timeline Estimator menggunakan tabel referensi historis untuk menyusun ringkasan timeline (skala proyek, durasi per fase, total durasi, resource per role) sebelum timeline generation berjalan.
- Tegaskan bahwa total durasi dapat berbeda dengan jumlah durasi per fase karena adanya fase paralel, bertumpuk, atau serial.
